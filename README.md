# MAS Frontend

MAS의 Frontend 코드입니다. MUI의 Template과 Next.js를 이용해 구현했습니다.

## 실행

### 실행 환경

---

- [install](https://nodejs.org/ko/download/) nodeJS 16.4.0 ver 이상

### 실행 방법

---

- clone git
<pre>
<code>
git clone https://github.com/Meeting-Auto-Summarization/frontend.git
git checkout deploy
</code>
</pre>
- npm install
<pre>
<code>
npm install
</code>
</pre>
- run
<pre>
<code>
npm run build
npm run start
</code>
</pre>
